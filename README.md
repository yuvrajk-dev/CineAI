# CineAI

CineAI is an AI-powered movie discovery platform built with React, Redux Toolkit, TMDB API, and Gemini AI.

Users can browse trending, popular, top-rated, and upcoming movies, explore detailed movie information, watch trailers, and get personalized movie recommendations through natural language prompts.

## Features

- AI-powered movie recommendations
- Trending, Popular, Top Rated, and Upcoming movies
- Movie search with TMDB API
- Detailed movie information and trailers
- Responsive design
- Dark and Light mode support

## Tech Stack

- React
- Redux Toolkit
- React Router
- Tailwind CSS
- TMDB API
- Gemini AI
- Vite
